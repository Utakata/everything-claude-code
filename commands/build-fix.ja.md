# ビルドと修正

TypeScriptとビルドのエラーを段階的に修正する：

1. ビルド実行: `npm run build` または `pnpm build`

2. エラー出力をパース:
   - ファイルごとにグループ化
   - 重大度でソート

3. 各エラーについて:
   - エラーのコンテキストを表示 (前後5行)
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラー解決を検証

4. 以下の場合に停止:
   - 修正が新しいエラーを引き起こした場合
   - 同じエラーが3回試行しても解決しない場合
   - ユーザーが一時停止を要求した場合

5. 概要を表示:
   - 修正されたエラー
   - 残っているエラー
   - 新たに導入されたエラー

安全のため、一度に1つのエラーを修正すること！
